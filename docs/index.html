kienzlezeit – die lokale Zeiterfassung für kleine Teams

Karte vorhalten, Zeit buchen, fertig. Arbeitszeiten, Anwesenheiten, Urlaub und Auslagen bleiben übersichtlich im eigenen Netzwerk.

GitHub Installer

Ein rundes RFID-Terminal für den Alltag – mit nachvollziehbaren Buchungen, klarer Teamplanung und vollständiger Administration im Browser.

Installation und Voraussetzungen

Einfach im eigenen Netzwerk

kienzlezeit läuft auf einem lokalen Debian- oder Ubuntu-System mit Apache, PHP und SQLite – sogar auf einem Raspberry Pi. Als Terminal dient ein M5Stack M5Dial mit RFID-Leser.

Bootstrap-Installer

Der Installer lädt die benötigten Anwendungsdateien direkt aus dem GitHub-Repository und richtet Apache, PHP und SQLite ein.

curl -fsSL \
  https://raw.githubusercontent.com/thomaskien/kienzlezeit/main/installer.sh \
  -o installer.sh
chmod +x installer.sh
sudo ./installer.sh
Installer ansehen →

Backup dringend empfohlen: Die SQLite-Datenbanken und die Schlüsseldatei sollten regelmäßig gemeinsam und auf einem zweiten Datenträger gesichert werden.

Technische Einordnung

Schlank, lokal und nachvollziehbar

  • M5Stack M5Dial für RFID-Buchungen direkt am Eingang
  • PHP und Apache für die browserbasierte Anwendung
  • SQLite für lokale, einfach sicherbare Datenhaltung
  • Raspberry Pi geeignet für einen sparsamen Dauerbetrieb
  • Auditprotokoll und Rohbuchungen für nachvollziehbare Änderungen
  • PDF- und CSV-Exporte für Monatsauswertungen
  • Optionales Auslagenmodul für Belege und Kilometer
